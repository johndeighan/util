git
===

1. Initialize your folder with: `git init`
2. Use `git status` to see what would be committed
	- adjust file `.gitignore` to adjust
3. Commit everything with:

```bash
$ git add -A
$ git commit -m "initial commit"
```

Create a new repo on github.com
NOTE: Use a standard windows Command Prompt to get good colors

```bash
$ gh repo create
```
select "Push an existing local repository to github.com"
Create a remote and name it 'origin'
